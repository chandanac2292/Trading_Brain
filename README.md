# Trading Brain 📈

A personal trading dashboard built for my friend using HTML, CSS, and JavaScript.

## Features

* Market quick access (NSE, BSE, MCX)
* Pre-trade checklist
* Position size calculator
* Trade log & P&L tracker
* News & screeners integration

## Tech Stack

* HTML
* CSS
* JavaScript
